### Prerequisites

* Python 3.8+
* `nmap` installed â€” [Get Nmap](https://nmap.org/download.html)
* **Linux/macOS:** `arp-scan` via your package manager
* **Windows:** [Npcap](https://npcap.com/) required for low-level scans

### Setup Steps

1. Clone the project:

   ```bash
   git clone https://github.com/kartik2005221/Omni-Scanner.git
   cd Omni-Scanner
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the scanner:

   ```bash
   sudo python3 main.py
   ```

   OR (on Windows):

   ```bash
   python main.py
   ```
